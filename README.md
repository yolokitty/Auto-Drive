• **자율주행 RC카 시스템**

고속 주행 환경에서 초음파 센서를 활용하여 장애물을 감지하고, 이를 기반으로 자율적으로 주행하는 RC카를 구현하였다.
초기에는 Ultrasonic sensor의 인식 속도 한계와 측정 오차로 인해 오작동이 발생하고, 선회 시 후미 간섭 문제가 발생하였다.
이를 해결하기 위해 센서 데이터 처리 로직을 개선하고 주행 알고리즘을 보완하여 안정적인 고속 주행 성능을 확보하였다.

---

**세부 기능**

RC카 기본 구동 구현 (전진 / 후진 / 좌우 회전)
→ 모터 제어를 통해 다양한 주행 동작 수행 가능

초음파 센서를 활용한 장애물 감지
→ 일정 거리 내 장애물 접근 시 거리 측정 및 판단

자율주행 알고리즘 구현
→ 장애물 위치에 따라 회피 경로를 판단하여 자동 주행 수행

---

**문제점 및 개선**

초음파 센서의 응답 속도 한계 및 노이즈로 인한 거리 측정 오차 발생
→ 필터링 및 데이터 보정 로직을 적용하여 신뢰도 향상

고속 주행 시 장애물 감지 지연으로 인한 충돌 위험 존재
→ 센서 샘플링 주기 최적화 및 예측 기반 제어 적용

선회 시 후미 간섭으로 인해 장애물과 충돌 발생
→ 회전 반경을 고려한 주행 알고리즘으로 보완
